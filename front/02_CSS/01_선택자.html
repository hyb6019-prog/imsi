선택자

HTML 문서 내부에서 스타일을 적용하고 싶은 요소(HTML 태그)를 선택할 때 사용하는 기술
선택자로 요소를 선택한 후 원하는 스타일을 적용

전체 선택자

HTML 문서 안의 모든 태그를 선택할 때 사용하는 선택자

 * {속성 : 값;}
ㅋㅋㅋㅋ

태그 선택자

HTML 문서 안의 동일한 태그를 모두 선택할 때 사용하는 선택지

 태그이름{속성 : 값;}

아이디 선택자

HTML 문서 안에서 해당하는 id속성을 가진 태그만 선택할 때 사용하는 선택자(id속성값은 중복x 해야만 함)

 #아이디{속성 : 값;}

  • 아이디 선택자 테스트2
  • 아이디 선택자 테스트3
  • 아이디 선택자 테스트4
  • 아이디 선택자 테스트5

클래스 선택자

HTML 문서 내에 여러 태그를 동시에 선택하고 싶은 경우 동일 클래스로 지정해서 선택

 .클래스이름{속성 : 값;}

  • 클래스 선택자 테스트3
  • 클래스 선택자 테스트4
  • 클래스 선택자 테스트5
