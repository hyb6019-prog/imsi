테두리 스타일

border-style : 테두리 스타일 지정

선택자{border-[방향]-style : none | hidden | dashed | dotted | double | groove | inset | outset | ridge | solid}

border-width : 테두리 두께 지정

선택자{border-[방향]-width : 두께 px}

border-color : 테두리 색상 지정

선택자{border-[방향]-color : 색상값;}

border : 테두리 스타일 한번에 지정

선택자{border-[방향] : width style color}

border-radius : 박스 모서리를 둥글게 하는 속성

선택자{border-[방향]-radius : 크기;}

box-shadow : box 그림자 주는 속성

선택자{box-shadow : 수평거리 수직거리 흐림 번짐 색상}

여백 스타일

margin : 요소와 요소 사이의 거리

    선택자{margin-[위치] : 거리;} -> top, bottom, left, right
    선택자{margin : 거리} -> 4방향 한번에 설정
    선택자{margin : 거리1 거리2} -> 거리1 : top/bottom, 거리2 : left/right
    선택자{margin : 거리1 거리2 거리3 거리4} -> 거리1 : top, 거리2 : right, 거리3 : bottom, 거리4 : left

padding : 테두리와 컨텐츠 사이의 거리

    선택자{padding-[위치] : 거리;} -> top, bottom, left, right
    선택자{padding : 거리} -> 4방향 한번에 설정
    선택자{padding : 거리1 거리2} -> 거리1 : top/bottom, 거리2 : left/right
    선택자{padding : 거리1 거리2 거리3 거리4} -> 거리1 : top, 거리2 : right, 거리3 : bottom, 거리4 : left
